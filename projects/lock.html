Digital Sequence Recognition Lock

Purdue ECE 270 | Digital Logic Design

Project Overview

Designed and implemented a modular digital lock system in SystemVerilog. The system accepts an 8-bit binary sequence from user pushbuttons and transitions through various states (INIT, LS0-LS7, OPEN, ALARM) to manage security[cite: 5, 31].

System Architecture

The design is strictly modular, consisting of five core submodules integrated into a top-level module[cite: 6, 7]:

  • Finite State Machine (FSM): Manages lock states and transition logic[cite: 11, 31].
  • Sequence Shift Register: Stores the 8-bit user-defined password[cite: 10, 27].
  • Display Module: Drives 7-segment displays for "SeCuRE", "OPEN", and "CALL 911" messages[cite: 12, 44].
  • Keypress Synchronizer: Debounces and synchronizes asynchronous button inputs[cite: 9, 86].
  • Clock Prescaler: Generates a 2 Hz signal for alarm flashing[cite: 8, 13].
