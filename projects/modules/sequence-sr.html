Sequence Shift Register (sequence_sr)

The shift register stores the 8-bit password sequence entered by the user via pushbuttons[cite: 65, 92].

Functionality

  • Serial Input: Accepts a 1-bit input representing the button pressed (0 or 1).
  • Enable Control: Only updates the internal sequence when the enable (en) signal is high, preventing accidental changes after the lock is set[cite: 47, 66, 94].
  • Parallel Output: Provides the full 8-bit sequence to the FSM for continuous comparison.

Implementation Code


always_ff @(posedge clk or posedge rst) begin
    if (rst)
        seq <= 8'b0;
    else if (en)
        seq <= {seq[6:0], button}; // Serial shift logic
end
