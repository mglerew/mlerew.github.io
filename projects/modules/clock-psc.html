Clock Prescaler (clock_psc)

This module generates a slower, prescaled clock from a high-speed input (100 Hz), specifically used to drive the 2 Hz alarm flash[cite: 45, 95].

Technical Implementation

  • Counter Logic: Implements an 8-bit counter that increments until it reaches a defined limit (lim)[cite: 51, 53].
  • Toggling Mechanism: When the count reaches the limit, the output signal toggles, effectively dividing the frequency[cite: 53, 98].
  • Passthrough Mode: If the limit is set to 0, the module passes the original clock through without division[cite: 54, 100].

SystemVerilog Snippet


module clock_psc(
    input  logic clk,
    input  logic rst,
    input  logic [7:0] lim,
    output logic hzX
);
    logic [7:0] tick_count;
    logic toggled_clk;

    always_ff @(posedge clk or posedge rst) begin
        if (rst) begin
            tick_count  <= 8'd0;
            toggled_clk <= 1'b0;
        end else begin
            if (lim != 8'd0) begin
                if (tick_count == lim) begin
                    tick_count  <= 8'd0;
                    toggled_clk <= ~toggled_clk;
                end else begin
                    tick_count <= tick_count + 8'd1;
                end
            end
        end
    end

    assign hzX = (lim == 8'd0) ? clk : toggled_clk;
endmodule
